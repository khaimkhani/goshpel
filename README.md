# goshpel
### a simple Go command line interpreter

## Status
#### This is currently a (working) work in progress.

## Installation
#### Build directly from repo

## Usage
```
$ goshpel // currently go run *.go from dir
>> import "fmt"
>> func foo(times int) {
...   for i := range times {
...     fmt.Println("bar")
...   }
...}
>> x := 3
>> foo(x)
bar
bar
bar
```
